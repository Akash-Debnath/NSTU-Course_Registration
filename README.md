# NSTU Course Registration

```
This is a web application built for Noakhali Science and Technology University course registration program. We know in every educational institution 
take fee from students anually or periodically. So, its a hassle to fillup large form, go to correspond person to take sign and money deposit in 
particular bank. So the system is made for eliminates all of the problems.
```

```
Languages : PHP, HTML, CSS, Bootstrap, JavaScript, Ajax, MySQL.
```
